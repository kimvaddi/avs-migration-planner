# Screenshot Guide

Save your screenshots here with these filenames:

| Filename | What to capture |
|----------|----------------|
| `dashboard.png` | Full Migration Dashboard tab (after import) |
| `import.png` | The import success notification bar |
| `ai-chat.png` | @avs /analyze in Copilot Chat panel |
| `dashboard-detail.png` | Dashboard zoomed into cost table + wave plan |
| `sidebar.png` | Activity Bar with VM Inventory + Recommendations trees |
| `bicep.png` | Generated Bicep template output |

## How to take screenshots

1. Press F5 in VS Code to launch Extension Development Host
2. Import your RVTools CSV
3. Use `Win+Shift+S` to capture each area
4. Save directly to this folder
